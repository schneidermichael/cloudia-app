## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ yarn run start

# watch restart including docker compose
$ yarn start:dev:restart

# watch mode
$ yarn start:dev

# production mode (todo)
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e pretest inludeing docker compose
$ yarn pretest:e2e

# e2e tests
$ yarn test:e2e

# test coverage (todo)
$ yarn test:cov
```
